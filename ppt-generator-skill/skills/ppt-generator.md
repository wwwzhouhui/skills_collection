# PPT生成技能 - 基于商务模板风格

## 技能说明
这个技能基于11.pptx、22.pptx、33.pptx三个模板的风格和结构，生成专业的商务PPT演示文稿。

## 模板特点分析

### 结构特点（固定25页）
1. **第1页**: 封面页 - 主标题、副标题、装饰元素
2. **第2页**: 目录页 - CONTENTS，列出4个主要章节
3. **第3页**: 第一章节过渡页
4. **第4-7页**: 第一章节内容页（4页）
5. **第8页**: 第二章节过渡页
6. **第9-12页**: 第二章节内容页（4页）
7. **第13页**: 第三章节过渡页
8. **第14-17页**: 第三章节内容页（4页）
9. **第18页**: 第四章节过渡页
10. **第19-22页**: 第四章节内容页（4页）
11. **第23页**: 结束页 - "谢谢观看"
12. **第24页**: 字体说明页
13. **第25页**: 版权声明页

### 设计风格特点
- **装饰元素**: 大量使用FREEFORM自由形状、AUTO_SHAPE自动形状、组合形状
- **文本要求**: 每页控制在200字以内，简洁生动
- **字体规范**:
  - 中文：阿里巴巴普惠体 2.0 55 Regular
  - 英文/数字：HarmonyOS Sans SC / MiSans Heavy
  - 标题：思源宋体 CN Heavy / Alimama ShuHeiTi Bold
- **色彩风格**:
  - 11.pptx: 年终总结风格，使用暖色调装饰
  - 22.pptx: 工作述职风格，商务简约
  - 33.pptx: 莫兰迪色系流体几何风

### 内容页布局类型
1. **双栏布局**: 左右分栏，各有标题和内容
2. **流程图布局**: 展示步骤或流程
3. **数据可视化**: 百分比、图表、数字展示
4. **列表布局**: 多个要点并列展示
5. **图文混排**: 图片+文字说明

## 使用说明

当用户提供PPT需求时，请按以下步骤操作：

### 步骤1：需求收集
向用户询问以下信息（如果未提供）：
1. PPT主题/标题
2. 副标题或适用场景（如：工作总结 / 工作汇报 / 述职报告 / 工作计划）
3. 4个主要章节的名称
4. 每个章节的关键内容要点（3-5个）
5. 是否需要数据可视化（图表、百分比等）
6. 色彩偏好（暖色调/商务简约/莫兰迪色系）

### 步骤2：生成PPT结构
使用python-pptx库创建PPT，按照以下结构：

```python
from pptx import Presentation
from pptx.util import Inches, Pt
from pptx.enum.text import PP_ALIGN
from pptx.dml.color import RGBColor

# 创建演示文稿
prs = Presentation()
prs.slide_width = Inches(16)  # 12192000 EMU
prs.slide_height = Inches(9)  # 6858000 EMU

# 添加25页幻灯片
```

### 步骤3：内容填充规范

**封面页（第1页）**：
- 主标题：居中，大字体
- 副标题：场景描述
- 年份或日期
- 装饰形状元素

**目录页（第2页）**：
- 标题："目录" / "CONTENTS"
- 列出4个章节标题
- 编号：01、02、03、04

**过渡页（第3/8/13/18页）**：
- 章节标题
- 章节编号
- 简短描述（200字以内）
- 装饰元素

**内容页（第4-7/9-12/14-17/19-22页）**：
- 页面标题
- 2-4个内容要点
- 每个要点包含：
  - 小标题（关键词）
  - 详细描述（50-100字）
- 可选：数据可视化元素

**结束页（第23页）**：
- "谢谢观看" / "Thank You"
- 副标题重复
- 联系方式（可选）

**字体说明页（第24页）**：
```
中文：阿里巴巴普惠体 2.0 55 Regular
英文 / 数字：HarmonyOS Sans SC / MiSans Heavy
标题：思源宋体 CN Heavy
```

**版权声明页（第25页）**：
```
仅供学习参考之用
```

### 步骤4：样式设置

**文本样式**：
- 主标题：36-48pt，加粗
- 章节标题：32-40pt，加粗
- 内容标题：24-28pt
- 正文：16-20pt
- 行间距：1.2-1.5倍

**颜色方案**（根据用户选择）：
- 暖色调：#FF6B6B, #FFD93D, #6BCB77
- 商务简约：#2C3E50, #3498DB, #95A5A6
- 莫兰迪色系：#B4A7A7, #A8DADC, #F1FAEE, #E63946

**形状装饰**：
- 使用圆角矩形、圆形、线条
- 渐变填充
- 半透明效果

### 步骤5：导出文件
- 文件名：根据主题命名，如 "2025年终工作总结.pptx"
- 保存位置：当前目录

## 示例用法

**用户输入**：
```
生成一个2025年度工作总结PPT，包含：
1. 年度概况
2. 重点项目回顾
3. 数据成果展示
4. 明年工作计划
```

**AI操作**：
1. 确认信息齐全
2. 创建25页PPT
3. 填充内容：
   - 封面："2025年度工作总结"
   - 目录：4个章节
   - 各章节按模板布局填充
   - 添加装饰元素
4. 保存文件

## 注意事项

1. **严格遵循25页结构**，不增不减
2. **文本简洁**：每页不超过200字
3. **视觉平衡**：文字与装饰元素比例适当
4. **字体一致性**：全文使用统一字体
5. **专业性**：避免花哨元素，保持商务风格
6. **可编辑性**：生成的PPT应便于后期修改
