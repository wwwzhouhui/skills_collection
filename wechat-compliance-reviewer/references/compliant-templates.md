# 合规话术模板

本文档提供微信公众号文章的合规话术模板，帮助作者规避违规风险。

---

## 一、替代违规话术

### 1.1 "免费/0 元"替代方案

| 违规表述 | 合规替代 |
|---------|---------|
| 完全免费 | 免费体验 / 新用户免费 |
| 0 元获取 | 首月免费体验 |
| 不要钱 | 无需付费（首月） |
| 白嫖 | 免费试用 / 优惠获取 |
| 薅羊毛 | 优惠活动 / 福利 |

**使用示例**：
- ❌ "0 元领取价值 50 美元的 Pro 会员"
- ✅ "首月免费体验价值 50 美元的 Pro 会员（需绑定支付方式，可随时取消）"

## 三、链接处理规范

### 3.1 允许放置的链接

| 类型 | 是否允许 | 说明 |
|------|---------|------|
| 微信公众号文章 | ✅ | 公众号群发内容 |
| 微信小程序 | ✅ | 小程序卡片 |
| 企业微信 | ✅ | 企业微信联系 |
| 腾讯系产品 | ⚠️ | 部分允许 |
| 外部网站 | ❌ | 原则上不允许 |

### 3.2 合规表述

| 场景 | 合规表述 |
|------|---------|
| 引用来源 | "来源：XXX 公众号" |
| 推荐阅读 | "推荐阅读：公众号 XXX 的《XXX》" |
| 官网信息 | "可在官网查看详细信息"（不放链接） |

## 四、图片使用规范

### 4.1 安全图片来源

- 自己拍摄的照片
- 购买授权的图库（Shutterstock、Getty 等）
- 免费可商用图库（Unsplash、Pexels 等）
- 官方提供的宣传素材

### 4.2 高风险图片

- 明星/名人肖像（需授权）
- 品牌 LOGO（需授权）
- 影视作品截图（可能侵权）
- 他人原创图片（需授权）

### 4.3 合规表述

```
【图片来源】
- 图 1-3：作者拍摄
- 图 4-6：来源于 Unsplash（可商用）
- 图 7-8：XXX 官方提供
```

## 五、完整示例

### 示例：技术教程文章合规版

```markdown
# 标题：AI 工具使用教程：XXX 平台上手指南

【声明】本文仅为技术分享，不构成任何购买建议。

## 前言

最近体验了 XXX 平台，感觉功能比较丰富，今天和大家分享一下使用方法。

## 平台介绍

XXX 是一个集成了多种 AI 模型的平台，支持...

【费用说明】
- 免费版：提供基础功能
- 付费版：$50/月，提供更多额度和高级功能
- 新用户可享受首月免费体验（需绑定支付方式）

## 使用步骤

### 1. 注册账号

访问官方网站，使用邮箱或第三方账号注册。

### 2. 开通服务

注册后可使用基础功能。如需更多额度，可升级付费版本。

【注意】升级需绑定支付方式，请仔细阅读取消订阅政策。

### 3. 获取 API Key

在设置页面可以生成 API Key，用于第三方工具接入。

## 使用体验

经过实际测试，平台的响应速度还不错，模型质量也比较好。

【个人感受】
- 优点：模型丰富、接口统一
- 注意：需要绑卡、有自动续费

## 总结

总体来说，XXX 平台适合需要多种 AI 模型的用户。

【建议】
1. 先试用免费版本，确认符合需求
2. 如需升级，了解清楚取消政策
3. 理性消费，按需购买

---

【温馨提示】
- 本文仅为技术分享
- 具体服务规则以官方为准
- 使用前请仔细阅读服务条款
- 投资/付费需谨慎决策
```

## 六、自查清单

发布前检查以下项目：

### 内容检查
- [ ] 没有"薅羊毛"、"白嫖"等词汇
- [ ] 没有"速转"、"分享"等诱导
- [ ] 没有"0 元"、"完全免费"等误导
- [ ] 没有"最强"、"第一"等绝对化用语
- [ ] 没有优惠码、推荐码
- [ ] 没有外部链接（或已处理）

### 声明检查
- [ ] 有技术分享声明
- [ ] 有费用说明（如适用）
- [ ] 有风险提示（如适用）
- [ ] 有版权声明（如适用）

### 图片检查
- [ ] 图片来源合法
- [ ] 已标注图片来源
- [ ] 无未授权品牌 LOGO
- [ ] 无未授权人物肖像

## 七、特殊行业注意事项

### 7.1 金融类

- 需要相关资质
- 不能推荐具体产品
- 不能承诺收益
- 必须提示风险

### 7.2 医疗健康类

- 需要相关资质
- 不能宣称治疗效果
- 不能传播偏方
- 必须科学依据

### 7.3 教育类

- 不能承诺效果
- 不能虚假宣传师资
- 不能编造学员案例
- 需注明培训机构资质

### 7.4 食品类

- 不能宣称保健功能
- 不能涉及疾病治疗
- 需要食品经营许可证
- 图片需与实物一致
