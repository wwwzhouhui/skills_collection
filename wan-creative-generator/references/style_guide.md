## 1. 坐标蓝图风格 🧪

### 核心特征
- 坐标系统：X/Y 轴线条，刻度标注
- 技术网格：均匀分割的网格背景
- 精密标注：数值、参数、箭头指向

### 色彩规范
- 主色：科技蓝 `#0A84FF`
- 辅色：数据绿 `#30D158`
- 背景：深空蓝 `#00162C`

### 构图模板
```
┌─────────────────────────────────────┐
│  Y轴                                │
│  ↑                                  │
│  │    [主体元素]                    │
│  │    ↑ 数据标注                    │
│  ├─────────────────────→ X轴        │
│  刻度线                              │
└─────────────────────────────────────┘
```

### Prompt 关键词
`坐标系统，技术网格，精密刻度，数据标注，科技蓝调，冷光高光，精密感，专业评测风格`

## 11. 构图类型详解

| 构图类型 | 特征 | 适用场景 |
|----------|------|----------|
| 特写聚焦型 | 主体放大居中，环境简化 | 产品展示、核心概念、单品突出 |
| 场景沉浸型 | 环境叙事，氛围营造 | 生活场景、故事感、空间展示 |
| 抽象隐喻型 | 符号化、概念化表达 | 抽象概念、品牌理念、隐喻传达 |
| 故事叙述型 | 情节感、时间线 | 过程展示、叙事内容、序列呈现 |
| 数据信息型 | 信息图表、数据可视 | 数据展示、对比分析、参数说明 |
| 对比冲击型 | 双元素对比、冲突感 | 对比评测、选择推荐、正反对比 |
| 拼贴组合型 | 多元素 collage | 多维度展示、合集、多样性表达 |
| 空间透视型 | 纵深层次、空间感 | 环境展示、场景透视、层次丰富 |

## 13. 创意生成模板（按起点类型）

### 具体物品模板

```
用户起点：[物品名称]

核心关键词：
- 形态：[形状、尺寸特征]
- 材质：[材料质感]
- 功能：[使用场景]
- 颜色：[主色调]
- 文化符号：[情感/象征意义]

发散路径：
1. 物品本身：放大/变形/重构/解构
2. 物品场景：使用环境/关联物品/人物互动
3. 物品隐喻：象征意义/符号化/抽象表达
4. 物品故事：叙事场景/时间线/情节感
5. 物品组合：与其他元素的碰撞/融合
```

### 抽象概念模板

```
用户起点：[概念名称]

核心关键词：
- 隐喻物：[可视化的代表物]
- 情绪：[情感调性]
- 场景：[发生的空间]
- 动作：[体现的行为]
- 符号：[抽象符号化]

发散路径：
1. 隐喻具象化：将抽象转化为具体视觉
2. 情绪氛围：用色彩/光影营造情感
3. 场景演绎：概念发生的空间叙事
4. 符号表达：抽象图形/几何化呈现
5. 对比冲突：概念的反面/张力呈现
```

### 生活场景模板

```
用户起点：[场景描述]

核心关键词：
- 空间：[物理环境]
- 人物：[参与者]
- 动作：[核心行为]
- 情绪：[情感状态]
- 标志细节：[标志性元素]

发散路径：
1. 空间重构：环境艺术化/超现实化
2. 人物聚焦：人物特写/情绪捕捉
3. 细节放大：标志性元素放大叙事
4. 情绪渲染：氛围营造/光影情感
5. 场景延伸：时间线/前后关联/变形
```
