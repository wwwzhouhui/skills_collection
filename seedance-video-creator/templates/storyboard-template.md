# Seedance 2.0 分镜提示词模板

> **重要**：Seedance 2.0 **必须至少提供一张参考图片**。没有用户图片时，需先生成首帧图片提示词，通过文生图 API 生成首帧参考图，再用于视频生成。

---

## 提示词结构说明

每个分镜需要生成 **两种提示词**：

### A. 首帧图片提示词（用于文生图生成首帧参考图）
- 描述一个**静止的画面瞬间**（不含动作、运镜等动态描述）
- 包含：场景、人物/主体外观、光影、色调、构图
- 画面比例应与视频比例一致
- 简洁精炼，100-200 字

### B. 视频分镜提示词（用于 Seedance 视频生成）
- 完整的分镜描述，用 `@1` 引用首帧参考图
- 包含时间轴、镜头语言、动作描述、声音设计
- 总长度控制在 500 字以内

---

## 模板一：叙事故事类

**首帧图片提示词**：
```
[风格]，[场景描述]，[主体外观和姿态]，[光影和色调]，[构图]
```

**视频分镜提示词**：
```
【风格】电影级写实/动画/水墨/科幻风格
【时长】4-15秒
【画幅】16:9 / 9:16 / 4:3

@1 作为画面首帧参考

0-X秒：[镜头运动]，[场景建立]，[主体引入]
X-Y秒：[镜头运动]，[情节发展]，[动作描述]
Y-Z秒：[镜头运动]，[高潮/冲突]，[情绪爆发]
...

【声音】配乐风格 + 音效 + 对白
```

## 模板二：产品展示类

**首帧图片提示词**：
```
[广告风格]，[产品特写/全景]，[材质和光影]，[背景和氛围]
```

**视频分镜提示词**：
```
【风格】商业广告/极简/高端/科技感
【时长】4-15秒

@1 作为产品首帧参考

0-X秒：产品展示，运镜环绕/推拉
X-Y秒：产品细节特写，材质/工艺展示
Y-Z秒：品牌落版，slogan展示

【声音】大气恢宏/轻快时尚/科技感配乐
```

---

## 模板三：角色动作类

**首帧图片提示词**：
```
[风格]，[角色全身/半身外观]，[服装和武器/道具]，[站姿/起始动作]，[场景和光影]
```

**视频分镜提示词**：
```
【风格】根据角色设定（武侠/科幻/现代/奇幻）
【时长】4-15秒

@1 作为角色参考

0-X秒：角色亮相，定格或缓慢展示造型
X-Y秒：核心动作展示（打斗/舞蹈/特技）
Y-Z秒：动作收尾，pose定格

【声音】动作音效 + 氛围配乐
```

---

## 模板四：风景旅拍类

**首帧图片提示词**：
```
[风格]，[大景别场景全貌]，[时间/季节/天气]，[色调和光影]，[构图]
```

**视频分镜提示词**：
```
【风格】电影级纪录片/治愈系/史诗感
【时长】4-15秒

@1 作为场景首帧参考

0-X秒：大景别建立镜头，展示环境全貌
X-Y秒：中景推进，引入人物或细节
Y-Z秒：回到大景别或意境落版

【声音】环境音 + 氛围配乐
```

---

## 模板五：视频延长/续拍

> 此模板需要用户提供已有视频截图作为参考图

```
@1 作为上一段视频的最后一帧参考

延续前视频的风格和主体：
0-X秒：[新内容描述]，与前视频无缝衔接
[继续描述新增内容的时间轴]

【要求】保持角色一致性，动作连贯流畅
```

---

## 模板六：视频编辑/改剧情

> 此模板需要用户提供原视频截图作为参考图

```
@1 作为原视频参考画面

【保留】原视频的运镜/部分动作/场景
【修改】[具体修改点1]
【修改】[具体修改点2]
【颠覆】[剧情反转描述]

【要求】保持镜头连贯，只在指定位置修改
```

## 常用镜头语言词汇表

| 中文 | 英文 | 用法 |
|------|------|------|
| 推镜头 | push in / zoom in | 强调主体，营造紧张感 |
| 拉镜头 | pull out / zoom out | 展现场景，营造疏离感 |
| 摇镜头 | pan | 左右/上下展示环境 |
| 移镜头 | dolly/truck | 平行移动，跟随主体 |
| 跟镜头 | follow | 跟随主体移动 |
| 环绕镜头 | orbit/circle | 360度展示主体 |
| 升降镜头 | crane up/down | 从上往下或从下往上 |
| 希区柯克变焦 | dolly zoom | 背景压缩/拉伸效果 |
| 手持晃动 | handheld | 纪实感/紧张感 |
| 一镜到底 | one shot | 不换镜头，连续拍摄 |

## 常用氛围描述词汇

**光影**：逆光、侧光、顶光、伦勃朗光、剪影、轮廓光、体积光、丁达尔效应
**色调**：暖色调、冷色调、高饱和、低饱和、黑白、赛博朋克、复古胶片
**质感**：电影级、纪录片风格、广告质感、MV风格、油画感、水墨感
**情绪**：温馨、紧张、悬疑、欢快、忧伤、史诗、治愈、惊悚

## 多模态引用格式

```
@1 作为首帧参考（首帧图片，由文生图生成或用户提供）
@2 作为尾帧/角色参考
@3-@6 作为场景/道具参考
```

注意：
- 提示词中的 `@N` 编号对应 API 调用时 `files` 参数中图片的上传顺序
- Seedance 2.0 **必须至少提供一张图片**（`@1`），否则返回错误
- 没有用户图片时，需先通过文生图 API 生成首帧参考图作为 `@1`
